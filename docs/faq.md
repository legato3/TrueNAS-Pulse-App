# Frequently Asked Questions

## What is TrueNAS Pulse?

TrueNAS Pulse is a native iOS application that provides a focused, operational interface for managing your self-hosted TrueNAS systems. It supports snapshot management, replication monitoring, cloud sync visibility, and manual task execution.

---

## Do I need a TrueNAS system to use the app?

Yes. TrueNAS Pulse connects directly to your own TrueNAS server. You configure the connection in the app using your TrueNAS API credentials. The app does not provide hosted TrueNAS services.

---

## Is my data sent to the developer or any third party?

No. TrueNAS Pulse does not collect, store, or transmit personal data. All communication occurs between your iOS device and your TrueNAS system. See the [Privacy Policy](../privacy/) for details.

---

## What is Guarded Mode?

Guarded mode is a safety feature that requires an extra confirmation step before sensitive actions (e.g., deletions, replications). It helps reduce accidental changes to your TrueNAS system.

---
